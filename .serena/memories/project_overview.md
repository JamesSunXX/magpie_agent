Magpie is a TypeScript/Node.js CLI for multi-model collaborative/adversarial engineering workflows. Main capabilities are review, discuss, trd, quality unit-test-eval, and loop. The codebase uses a mixed architecture: legacy command flows remain for review/discuss/trd, while loop and quality/unit-test-eval run through the V2 capability runner. Key top-level areas are src/cli, src/capabilities, src/core, src/platform, plus legacy folders like src/commands, src/orchestrator, src/providers, src/context-gatherer, src/reporter, and mirrored Vitest tests under tests/.
